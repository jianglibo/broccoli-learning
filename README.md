# Broccoli 简介

Broccoli作为一个构建工具[^1]不算很流行。但有时候流行的不一定是适合的，至少对某些人来说。

Broccoli的文档不是很丰富，由于版本更新的缘故，网上出现的示例代码不一定能运行。不管怎样，其简洁的设计能够让用户立马开始自己书写plugin，解决特定的问题。这个[项目](https://github.com/jianglibo/broccoli-learning)的托管在github上，因为gitbook和github的集成是如此自然，所以顺便将学习Broccoli的过程中学到的东西，写成这本gitbook。

[^1]: 在[博客](https://www.solitr.com/blog/2014/02/broccoli-first-release/)上面将作者将其称作一个新的构建工具(is a new build tool)，同时将broccoli和rubyonrails的asset pipeline作对比。我个人理解，应该是一个简单的构建工具，主要用于asset pipeline。
